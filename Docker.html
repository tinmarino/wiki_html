Command

  • Copy: docker cp foo.txt mycontainer:/foo.txt
  • List: docker ps -a

Run

  • -i : read from stdin
  • -t : prompt to terminal
  • -e VAR=value : set an environment variable before invoking the command
  • -d : run in detached mode
  • –name=toto : set the name of container to toto

Location

  • var/lib/docker
    • containers
    • image
    • volume
